# Work Completion Checklist

Repository: `StefanoCaruso456/chatbox-audit`

GitHub: `https://github.com/StefanoCaruso456/chatbox-audit`

Use this checklist every time a task is finished.

## Required steps

- Confirm the intended files are the only files changed.
- Commit the completed work with a clear message.
- Push the branch to GitHub right away.
- Open a pull request before calling the task done.
- Verify the remote branch contains the latest local work.

## Goal

Keep local work and GitHub in sync so the repository does not fall behind and merge conflicts stay manageable.
